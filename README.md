## About the project:

### Built with:
This project is built with Node.js and React, as well as Javascript and appropriate css. Express.js and MongoDB were also used

### App Description:
Stitched is a pre-loved thrifting online website, wherein users can post their second hand clothing and accessories for a little side cash

## Getting Started:

### Prerequisites:

* [Git](https://git-scm.com/downloads)
* [Node.js](https://nodejs.org/en)
* [NPM](https://www.npmjs.com/)

### How to install:

* React Bootstrap <br> `npm install react-bootstrap bootstrap`
* React Router DOM <br> `npm i react-router-dom`
* Bootstrap <br> `npm i bootstrap`
* Material UI <br> `npm install @mui/material @emotion/react @emotion/styled`
